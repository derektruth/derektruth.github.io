CreateConnection(string)

Creates a new connection to the database, adds it to the connection list, and ensures that the connection is removed from the list as soon as it is disposed. Will log a warning if the connection list reaches 20+ open connections.

note

This calls helps track all open connections and their callers which can help finding leaks.
